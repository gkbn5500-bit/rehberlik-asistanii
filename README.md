# Rehberlik Asistanı

Türk okulları için geliştirilmiş aylık rehberlik faaliyet raporu oluşturma uygulaması.

## Özellikler

* 5, 6, 7 ve 8. sınıf düzeylerini destekler.
* Sınıf ve hafta seçimine göre etkinlikleri otomatik getirir.
* Etkinlik adı, kazanım ve yeterlilik alanlarını otomatik doldurur.
* Aylık rehberlik faaliyet raporunu oluşturur.
* Yazdırma ve PDF olarak kaydetme desteği sunar.
* Mobil uyumludur.

## Amaç

Öğretmenlerin aylık rehberlik faaliyet raporlarını hızlı ve kolay şekilde hazırlamalarına yardımcı olmak.

## Kullanılan Teknolojiler

* HTML
* CSS
* JavaScript

## Gelecek Sürümler

* Veli görüşme formu
* Öğrenci görüşme formu
* Tutanak oluşturucu
* Karne görüşleri oluşturucu
* Rehberlik evrakları yönetim sistemi
